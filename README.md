# PureLily 17 Keywords

🌸 *The official programming language of the Cookie Kingdom*

---

## Control Flow

| Keyword | Usage | Description |
|---------|-------|-------------|
| `if` | `if condition { }` | Conditional branch |
| `else` | `if { } else { }` | Alternative branch |
| `match` | `match value { pattern => expr }` | Pattern matching |
| `when` | `match { pattern when guard => }` | Match guard condition |
| `return` | `return value` | Return from a function |
| `bloom` | `bloom target { }` | Awaken the bloom within |
| `wish` | `wish("string")` | Make a wish upon a star |
| `heal` | `heal(target, amount)` | Heal and restore |
| `iflight` | `iflight { }` | Illuminate the path forward |
| `sparkle` | `sparkle()` | Create magical sparkles |
| `glow` | `glow()` | Emit light glow |
| `shine` | `shine()` | Shine brightly |

## Loops

| Keyword | Usage | Description |
|---------|-------|-------------|
| `for` | `for item in items { }` | Iterate over items |
| `while` | `while condition { }` | Loop while condition holds |
| `repeat` | `repeat N times { }` | Repeat a block N times |
| `moonpath` | `moonpath { }` | Walk the moonpath (dream block) |

## Async / Temporal

| Keyword | Usage | Description |
|---------|-------|-------------|
| `async` | `async fn name() { }` | Async function modifier |
| `await` | `await expr` | Wait for a timeflow operation |
| `timeflow` | `timeflow { }` | Async block for temporal operations |

## Declarations

| Keyword | Usage | Description |
|---------|-------|-------------|
| `let` | `let name = value` | Declare a mutable variable |
| `const` | `const NAME = value` | Declare an immutable constant |
| `fn` | `fn name(params) -> Type { }` | Define a function |
| `class` | `class Name { }` | Define a class |
| `struct` | `struct Name { }` | Define a struct |
| `enum` | `enum Name { Variant }` | Define an enum |
| `trait` | `trait Name { }` | Define a trait (interface) |
| `impl` | `impl Name for Type { }` | Implement a trait for a type |
| `type` | `type Alias = Existing` | Type alias definition |
| `friend` | `friend name { }` | Declare a friendship bond |
| `dragon` | `dragon name { }` | Declare a dragon power block |
| `ancient` | `ancient name { }` | Access ancient-level abilities |

## Modules

| Keyword | Usage | Description |
|---------|-------|-------------|
| `import` | `import name from "path"` | Import a module |
| `export` | `export fn name()` | Export a symbol |
| `from` | `import { } from "path"` | Module source specifier |
| `mod` | `mod name { }` | Declare a submodule |
| `pub` | `pub fn name()` | Public visibility modifier |
| `super` | `super::name` | Parent scope reference |
| `self` | `self.method()` | Self reference |
