✨ What is PureLily 17?

PureLily 17 is a beautiful, powerful, beginner-friendly programming language that celebrates 17 years of Cookie Run. It features a complete compiler, package manager, registry, formatter, linter, LSP, IDE ecosystem, and magical developer experience.

🚀 Quick Start

# Create a new project
lily new my-kingdom

# Run your code
cd my-kingdom
lily run src/index.lily

# Format and lint
lily format src/
lily lint src/

# Build
lily build

# Start the Language Server
lily lsp start

🍪 Hello, Cookie Kingdom!

// 🎉 Celebrating 17 Years of Cookie Run!
